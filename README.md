## 📘Technology Stack

- Clinet: MFC, C++, MySQL

## 📌Summary

  **Tetris 게임은 첫 회사인 뉴크에 취업 후, MFC 스터디 용으로 개발한 다이얼로그 기반 게임입니다.**  테트리스 게임 개발 후, 레지스트리를 이용한 배경색 변경, DB를 활용한 로그인 기능, TCP통신 기능을 추가해보라는 조언에 따라 해당 기능들을 추가했습니다. 

 대표님의 컨펌을 받으며 OOP가 무엇인가 고민하며 개발한 어플리케이션입니다. C++의 기초는 알고 있었지만, MFC는 처음 다루어 봤기 때문에 회사 프로젝트 투입 전 도움이 많이 됐습니다. **MFC를 다루는 법을 익혔고, TCP 통신이 무엇인지, DB는 어떻게 다루는 것 인지에 대해서** 개발을 통해 처음 느꼈던 프로젝트였습니다. 

### 주요기능

- Timer기반의 블럭 좌표 이동
- 레지스트리를 이용한 배경색 변경
- 배경음악 재생, 정지 기능 구현
- MySQL 로그인, 회원가입 기능 구현
- TCP 통신 기반의 1대1 채팅 기능 구현
